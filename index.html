Security Incident Timeline — FY 2025–2026

Phishing Attack — Member Portal Impersonation
May 16, 2026 | Severity: High | Status: Contained

3 employees received spoofed emails mimicking member portal login. Credentials not captured; MFA prevented access. Emails blocked at gateway. Awareness alert issued org-wide.

Unauthorized API Access Attempt
Mar 4, 2026 | Severity: Medium | Status: Resolved

SIEM detected 2,400 failed API authentication attempts from 14 IP addresses. Rate limiting and IP block applied. No data accessed. Source attributed to credential stuffing botnet.

Ransomware Indicator — Endpoint Quarantine
Nov 19, 2025 | Severity: Critical | Status: Contained

EDR detected ransomware pre-execution signatures on contractor laptop. Endpoint isolated within 8 minutes. Forensic analysis confirmed no lateral movement. Contractor access revoked pending review.

Insider Threat — Unauthorized Data Access
Sep 8, 2025 | Severity: High | Status: Resolved

DLP alert triggered for bulk export of 1,400 member records by terminated employee whose access was not revoked within SLA. Access removed; records confirmed not exfiltrated externally. HR/Legal notified.

Web Application Vulnerability Exploited — XSS
Jul 22, 2025 | Severity: Medium | Status: Patched

Cross-site scripting vulnerability identified in public-facing document submission form. No PII exposed. Patch deployed same day. WAF rule added as interim control. Pen test findings closure verified.
